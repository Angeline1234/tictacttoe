# Tic Tac Toe Game

A simple, browser-based **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**.  
This project demonstrates core frontend concepts, including DOM manipulation, event handling, and basic game logic.

---

## Features

- Two-player gameplay (Player X vs Player O)
- Interactive 3×3 game board
- Win detection (rows, columns, diagonals)
- Draw detection
- Turn-based status updates
- Restart/reset functionality
- Clean and responsive layout

---

## Technologies Used

- **HTML** – Structure and layout of the game board  
- **CSS** – Styling, grid layout, and visual feedback  
- **JavaScript** – Game logic, state management, and DOM interaction

## How to Run the Game

1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Click on a cell to make a move
4. Use the **Restart** button to reset the game at any time

_No additional setup or dependencies required._

---

## Game Logic Overview

- The board state is stored as an array of 9 elements
- Each cell click updates the board and switches turns
- The game checks predefined winning combinations after every move
- The game ends when a player wins or when all cells are filled (draw)
